sidebarMenuListItem
directive in module sidebarMenu

Description

This directive fill manager sidebar item <li></li> html element with given item.

This shouln't be used outside the sidebarMenu module scope.

Usage

as attribute
<ANY sidebar-menu-list-item
     sidebar-menu-list-item="{SidebarMenuListItem}">
   ...
</ANY>

Parameters

ParamTypeDetails
sidebar-menu-list-itemSidebarMenuListItem

The menu item instance to display.
