sidebarMenu
directive in module sidebarMenu

Description

This is the base directive to load into your universe code. This directive will load the "root" items and will manage (with SidebarMenu service) sub items loading and display.

Basically, the directive will load the other module's directives (sidebarMenuMenu and sidebarMenuMenuItem).

Usage

as attribute
<ANY sidebar-menu>
   ...
</ANY>
