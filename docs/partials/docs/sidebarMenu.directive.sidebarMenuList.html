sidebarMenuList
directive in module sidebarMenu

Description

This directive fill manager sidebar <ul></ul> html element with given items.

This shouln't be used outside the sidebarMenu module scope.

Usage

as attribute
<ANY sidebar-menu-list
     sidebar-menu-list-items="{Array<SidebarMenuListItem>}">
   ...
</ANY>

Parameters

ParamTypeDetails
sidebar-menu-list-itemsArray<SidebarMenuListItem>

The items to be filled into list element of manager sidebar.
